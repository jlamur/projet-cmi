# projet-cmi

Projet CMI

Arborescence :

```
- docs : comptes-rendus, rapports et autres documents
- res : ressources (textures, images, musique)
- src : code source (.cpp)
- include : en-têtes (.hpp)
```

Compilation :

```
$ cd projet-cmi
$ cmake .
$ make
```
